# FreeCAD Python API Dependencies
# For manufacturing drawing generation

# FreeCAD is typically installed as a standalone application
# Python API is included with FreeCAD installation
# This file documents the setup requirements

# FreeCAD Installation:
# - Windows: Download from https://www.freecad.org/downloads.php
# - Linux: sudo apt-get install freecad (or use AppImage)
# - macOS: Download from https://www.freecad.org/downloads.php

# FreeCAD Python API Usage:
# - FreeCAD Python API is accessed via FreeCAD module
# - Run scripts using: freecadcmd script.py
# - Or import FreeCAD in Python: import FreeCAD
